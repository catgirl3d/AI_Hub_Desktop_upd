# AI Hub Desktop 

AI Hub Desktop is a desktop workspace that brings multiple AI providers into one streamlined workflow. Pick a provider, start a chat, and work in a focused window without jumping between browser tabs.

The app is built for quick iteration and easy sharing of results. When a session is ready, export or share the output, and keep the app updated with built-in notifications.

- Choose from multiple AI providers and switch sessions quickly.
- Keep a focused chat workflow with dedicated windows and navigation.
- Schedule and run deep-research sessions for longer tasks.
- Export or share conversations in common formats (including PDF).
- Receive update and notification prompts to stay current.
